## About chooseForMe

App where users can add options and it will randomly select one.

### Built With

* ReactJS
* NodeJS
* PostgreSQL


## Getting Started

### Prerequisites


* Download files and execute these commands in /backend and /frontend
  ```sh
  $ npm install
  ```
  ```sh
  $ npm start
  ```
